# obsidian-tasks-utils
Scalable script that integrates your Obsidian vault with Pushover's API for sending messages.

Can be used to fetch your daily tasks and show them via Scriptable as a widget on your iOS homescreen (for example).

### .env
```
PUSHOVER_API_TOKEN=
PUSHOVER_USER_KEY=
OBSIDIAN_VAULT_PATH=
HOST=
PORT=
```
